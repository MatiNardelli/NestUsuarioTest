# Team DB

1. Clonar proyecto
2. ```yarn install```
3. Clonar el archivo ```.env.tamplate``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno
5. Inicializar: ```yarn start:dev```

6. Levantar la base de datos
```
docker-compose up -d
```
